Å° motor
===================================================================================
motor1:
	forward
	backward
	stop
motor2:
	forward
	backward
	stop
===================================================================================
cmd usage:
	motor_action(motor1, motor2, motor1_act, motor2_act)
===================================================================================
protocol definition: 3 bytes
	------------------------------------------------------------------
	1st byte: cmd code, motor selection, motor action
		------------------------------------------------------------------
		0 - unused
		1 - unused
		------------------------------------------------------------------
		2 - with bit[3]
		3 - motor1_act: [00]stop; [01]forward; [11]backward
		------------------------------------------------------------------
		4 - with bit[5]
		5 - motor2_act: [00]stop; [01]forward; [11]backward
		------------------------------------------------------------------
		6 - with bit[7]
		7 - cmd type code: [01] for motor action
===================================================================================

Å° servo
===================================================================================
servo1: head
	140 degree rotate (70 degree each side)
servo2: left arm
	300 degree rotate
servo3: right arm
	300 degree rotate
===================================================================================
cmd usage:
	servo_action(servo1, servo2, servo3, servo1_deg, servo2_deg, servo3_deg)
===================================================================================
protocol definition: 4 bytes
	------------------------------------------------------------------
	1st byte: cmd code, servo selection
		------------------------------------------------------------------
		0 - servo1: [0]disable; [1]enable
		1 - servo2: [0]disable; [1]enable
		2 - servo3: [0]disable; [1]enable
		------------------------------------------------------------------
		3 - unused
		4 - unused
		5 - unused
		------------------------------------------------------------------
		6 - with bit[7]
		7 - cmd type code: [10] for servo action
	------------------------------------------------------------------
	2nd byte: servo1 rotate degree
		------------------------------------------------------------------
		[0-5] 6 bit for servo1 degree value:
			* max 63 scale
		------------------------------------------------------------------
		6 - with bit[7]
		7 - data type code: [00] for data
	------------------------------------------------------------------
	3rd byte: servo2 rotate degree
		------------------------------------------------------------------
		[0-5] 6 bit for servo2 degree value:
			* max 63 scale
		------------------------------------------------------------------
		6 - with bit[7]
		7 - data type code: [00] for data
	------------------------------------------------------------------
	4th byte: servo3 rotate degree
		------------------------------------------------------------------
		[0-5] 6 bit for servo3 degree value:
			* max 63 scale
		------------------------------------------------------------------
		6 - with bit[7]
		7 - data type code: [00] for data
===================================================================================


Å° LED
===================================================================================
LED1:
	on
	off
LED2:
	on
	off
===================================================================================
cmd usage:
	led_action(led1, led2, led1_color, led2_color)
===================================================================================
protocol definition: 5 bytes
	------------------------------------------------------------------
	1st byte: cmd code, LED selection
		------------------------------------------------------------------
		0 - LED1: [0]disable; [1]enable
		1 - LED2: [0]disable; [1]enable
		------------------------------------------------------------------
		2 - unused
		3 - unused
		4 - unused
		5 - unused
		------------------------------------------------------------------
		6 - with bit[7]
		7 - cmd type code: [11] for LED action
	------------------------------------------------------------------
	2nd byte: LED1 color
		------------------------------------------------------------------
		[0-5] 6 bit for LED1 color:
			* max 63 scale
		------------------------------------------------------------------
		6 - with bit[7]
		7 - data type code: [00] for data
	------------------------------------------------------------------
	3rd byte: LED2 color
		[0-5] 6 bit for LED2 color:
			* max 63 scale
		------------------------------------------------------------------
		6 - with bit[7]
		7 - data type code: [00] for data
===================================================================================
